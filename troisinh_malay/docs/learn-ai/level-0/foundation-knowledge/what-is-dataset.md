# Dataset adalah apa?

Dataset adalah set data yang disusun untuk komputer belajar. Fahami betul konsep ini memmemastikan anda pegang cara AI 'belajar' daripada realiti.

Copy MarkdownOpen

## [Definisi](what-is-dataset.html#definition)

Dataset adalah set data yang disusun mengikut struktur tertentu supaya komputer boleh asli, memproses dan belajar. Secara mudah, ini adalah "tambahan" atau "buku teks" yang AI guna untuk belajar cara laksanakan tugasan, dan bukan sekadar data mentah yang bercampur dalam pelayan.

## [Penjelasan terperinci](what-is-dataset.html#detailed-explanation)

### [Dataset beza dengan "data bercampur" biasa?](what-is-dataset.html#dataset-another-gì-with-dữ-liệu-lộn-xộn-thông-thường)

Data mentah (raw data) seperti log pelayan belum diproses, imej diambil telefon secara rawak, atau mesej WhatsApp yang disimpan adalah data tanpa struktur. Dataset adalah data yang telah dibersihkan, diberi label (label), dan disusun supaya komputer faham setiap hubungan.

Contoh: dan bukan 10,000 imej diambil secara rawak dalam komputer, Dataset adalah 10,000 imej yang telah diklasifikasikan dengan jelas sethirdgai "kucing" atau "anjing" untuk AI belajar cara mengenal.

### [Struktur satu Dataset](what-is-dataset.html#cấu-trúc-of-one-dataset)

Dataset selalunya berbentuk jadual (tabular) seperti Excel atau set fail khusus:
