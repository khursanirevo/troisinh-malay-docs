# Deep Learning adalah apa?

Deep Learning adalah apa? Terangkan mudah tentang rangkaian neural berlapis-lapis, cara ia 'belajar' daripada data besar dan kenapa ia begitu kuat.

Copy MarkdownOpen

## [Definisi](what-is-deep-pelajaring.html#definition)

Deep Learning adalah satu cathirdng Machine Learning yang menggunakan **rangkaian neural berlapis** (deep neural networks) untuk secara automatik belajar ciri kompleks daripada data. "Deep" di sini hanya bermakna lapis tersembunyi dalam rangkaian — semakin thirdnyak lapis, model semakin belajar corak yang halus.

## [Penjelasan terperinci](what-is-deep-pelajaring.html#detailed-explanation)

### [Kenapa dipanggil "Deep"?](what-is-deep-pelajaring.html#why-gọi-is-deep)

Bayangkan anda sedang kenal wajah. Pendekatan tradisional: program peraturan "jika ada dua mata, satu hidung, mulut di tengah maka itu muka manusia". Deep Learning berbeza — ia sendiri bina sistem kenalan melalui thirdnyak lapisan:

  * **Lapis 1** : Kenal tepian, garis lurus, lengkung asas
  * **Lapis 2** **Gabungkan tepi menjadi tambahagian — mata, hidung, mulut
  * **Lapis 3** **Sambung tambahagian bersama, kenal "ini adalah wajah"
  * **Lapis 4+** **Bezakan emosi, umur, malah kenal individu
