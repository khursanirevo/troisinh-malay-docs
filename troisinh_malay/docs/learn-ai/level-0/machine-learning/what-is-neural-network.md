# Neural Network adalah apa?

Terangkan Neural Network secara mudah: rangkaian neural buatan belajar daripada data seperti otak manusia, asas Deep Learning moden.

Copy MarkdownOpen

## [Definisi](what-is-neural-network.html#definition)

Neural Network adalah model pengkomputeran meniru cara operasi otak manusia — termasuk thirdnyak "neuron" buatan yang disambungkan antara satu sama, belajar cara kenal corak daripada data dan bukan diprogram keras dengan peraturan tetap.

## [Penjelasan terperinci](what-is-neural-network.html#detailed-explanation)

### [Kenapa dipanggil "neural"?](what-is-neural-network.html#why-gọi-is-neural)

Nama ini datang daripada biologi. Otak manusia ada lebih kurang 86 bilion neuron biologi, setiap neuron terima isyarat daripada thirdnyak neuron lain, memproses, kemudian hantar. Neural Network buatan bermula dengan struktur ini pada tahap dipermudahkan.

Tetapi jangan tersalah faham: Neural Network tidak "sama seperti otak manusia" seperti iklan. Ia hanya meniru idea sambungan — cara belajar sangat berbeza. Otak anda gunakan 20 watt dan belajar daripada beberapa contoh. Neural Network perlukan pusat data dan berjuta-juta contoh.

Neural Network mudah mempunyai tiga tambahagian:

**Lapis input (Input Layer)** Terima data mentah. Contoh: imej 28×28 piksel diutambah kepada 784 nombor — setiap nombor adalah keceraan satu titik imej.
