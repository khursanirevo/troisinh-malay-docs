# Prompt tulis tesis terperinci

Set prompt tulis tesis optimum untuk pelajar: daripada pilih topik ke edit gaya penulisan, memastikan siap skrip lebih pantas 50% tapi kekalkan sifat akademik.

Copy MarkdownOpen

Prompt tulis tesis adalah sistem arahan berstruktur digunakan untuk kawal AI main peranan "pemmemastikan penyelidikan" — memastikan analisis topik, rumuskan teori, edit logic dan gaya akademik — daripada tulis ganti pengguna.

## [Penjelasan terperinci](thesis-writing-prompts.html#detailed-explanation)

### [Bezakan "pemmemastikan" dan "magic"](thesis-writing-prompts.html#bezakan-pembantu-dan-magic)

Salah terbesar pelajar bila guna AI untuk tesis adalah anggap ini "penulis ajaib". Sebenarnya, AI tak boleh ganti anda berfikir, juga tak tanggungjawab atas data lapangan atau penemuan saintifik baru. Prompt betul letak AI macam "pemasli kritikal maya" — tunjuk luthirdng logic, cadang sudut pandang baru, bukan janakan kandungan asal ganti anda.

### [Set prompt ikut kitar hayat tesis](thesis-writing-prompts.html#set-prompt-ikut-kitar-hayat-tesis)

#### [Prompt teroka topik (Research Gap)](thesis-writing-prompts.html#prompt-teroka-topik-research-gap)

Guna bila anda belum ada topik spesifik:
    Saya berminat bidang [masukkan bidang, contoh: pemasaran digital di Malaysia]. 
    Senarai 5 isu penyelidikan masih ada jurang (research gap) sesuai untuk pelajar universiti, 
    boleh capai dengan kaedah kuantitatif melalui tinjauan. 
    Untuk setiap isu, nyatakan: (1) Isu spesifik, (2) Sehalaman masih ada jurang, (3) Kaedah cadangan.

#### [Prompt cadangan terperinci (Outline)](thesis-writing-prompts.html#prompt-cadangan-terperinci-outline)
    Cipta cadangan terperinci untuk topik "[yeara topik]" ikut struktur skrip sarjana muda 3 halaman:
    - Bab 1: Asas teori (perlukan petikan sekurang-kurangnya 3 penulis luar negara dan 2 kajian dalam negara)
    - Bab 2: Kaedah Penyelidikan (kuantitatif/gabung)
    - Bab 3: Hasil dan Perbincangan
    Syarat: Setiap halaman ditambahagikan ikut seksyen, subseksyen, dan nota gusaha kandungan perlukan tulis untuk setiap subseksyen.

#### [Prompt tinjauan dokumen (Literature Review)](thesis-writing-prompts.html#prompt-tinjauan-dokumen-literature-review)
    Saya ada 5 petikan pasal [topik]. Analisis dan banding:
    - [Petikan 1: Penulis - Tahun - Kesimpulan utama]
    - [Petikan 2...]
    Tugas:
    1. Kumpulkan kajian ikut pandangan (sahajakorang/neutral/menentang)
    2. Tunjuk percanggahan antara kajian
    3. Kenal pasti jurang yang tesis saya akan isi
    Nota: Cumalah analisis berdasarkan maklumat sediakan, tak reka tambah kajian.

#### [Prompt kritikan logic (Peer Review)](thesis-writing-prompts.html#prompt-kritikan-logic-peer-review)
    Main peranan pensyarah kritikan skrip. Baca segmen ni dan tunjuk:
    - Kelemahan pasal logic hujah (jika ada)
    - Kekurangan pasal bukti (perlukan tambah data atau petikan?)
    - Soalan boleh buat penulis bingung dalam pembelaan
    - Cadangkan tingkatkan struktur segmen
    Segmen: "[masukkan segmen 200-300 perkataan]"

#### [Prompt edit gaya akademik](thesis-writing-prompts.html#prompt-edit-gaya-akademik)
    Tukar segmen ni ke gaya penulisan akademik Bahasa Melayu, formal tapi mudah faham:
    - Ganti kosakata harian dengan istilah pakar [yeara bidang]
    - Guna struktur ayat kompleks, sambung ketat antara klausa
    - Kekalkan maksud asal, tak tambah maklumat baru
    - Elak guna proyeara pertama "saya", "kami" (tukar ke bentuk tanpa yeara atau pasif jika sesuai)
    Teks asal: "[masukkan teks]"

#### [Prompt periksa petikan](thesis-writing-prompts.html#prompt-periksa-petikan)
    Periksa senarai petikan ni patuh standard [APA/Harvard/Vancouver] tak. 
    Untuk setiap item, tunjuk ralat format (jika ada) dan cadang sepertii semula.
    Nota: Tak periksa ketepatan kandungan penyelidikan, cuma periksa format.
    Senarai:
    [masukkan senarai]

### [Teknik Context Injection](thesis-writing-prompts.html#teknik-context-injection)

Tesis mendalam minta AI faham konteks spesifik. Teknik ni minta anda sediakan cadangan terperinci, data sampel, dan gaya petikan permintaan fakulti/sekolah terus dalam Prompt.

Contoh struktur:
    Berdasarkan cadangan berikut: [masukkan cadangan]
    Dan data statistik: [masukkan jadual nombor]
    Tulis tambahagian 3.2 "Analisis hasil tinjauan" dengan andaian anda adalah penyelidik bidang [X].

### [Proses had dan ralat](thesis-writing-prompts.html#proses-had-dan-ralat)

AI selalunya "halusinasi" (hallucinate) petikan. Sentiasa tambah di hujung setiap Prompt penyelidikan:
    Nota: Cumaguna kajian saya sediakan. Jika tak pasti pasal satu maklumat, cakap "Perlukan semak" daripada reka.

## [Contoh sebenar](thesis-writing-prompts.html#real-world-examples)

**Pelajar Pemasaran kaji tingkah laku beli di TikTok Shop**

Razak adalah pelajar tahun 4, bidang Pemasaran, perlukan tulis pasal "Kesan konten pendek kepada keputusan beli impulsif Gen Z di TikTok Shop". Daripada habis 2 minggu cari teori, Razak guna Prompt tinjauan dokumen untuk banding model S-O-R (Stimulus-Organism-Response) dengan Theory of Planned Behavior, memastikan pilih rangka kerja teori sesuai untuk konteks Malaysia cuma dalam 2 jam.

**Pelajar Bahasa Inggeris analisis karya sastera**

Aina sedang tulis tesis banding "Sebuah Kapal di Tengah Lautan" (A. Samad Said) dengan sebuah cerpen pendek Rusia. Dia guna Prompt kritikan untuk periksa ketepatan pelbagai bandingan teks, memastikan kesan ralat perbandingan tak sepadan (false equivalence) dalam segmen analisis watak yang dia sendiri tulis.

**Pelajar IT terangkan algoritma**

Kumar, pelajar Sains Komputer, perlukan tulis tambahagian "Penerangan Sistem" untuk projek sarjana muda pasal Machine Learning. Dia guna Prompt tukar gaya penulisan untuk tukar nota kod mentah jadi penerangan akademik pasal seni bina neural network, pastikan orang tak tahu pengaturcaraan pun faham logic operasi.

## [Aplikasi](thesis-writing-prompts.html#applications)

**Pelajar universiti (Skrip Sarjana Muda)**

Gunakan set prompt untuk ringkaskan masa cari dokumen dan edit struktur. Fokus pada Prompt cadangan dan kritikan untuk elak tersasar muka — ralat popular menyehalamankan pindaan besar lepas panel pakar.

**Pelajar pascasiswazah (Tesis Sarjana)**

Guna Prompt tinjauan dokumen untuk bina tambahagian kajian teori ketat, minta sistematik dan bandingan antarathirdngsa tinggi. Gabung dengan periksa petikan untuk pastikan sifat akademik ketat ikut standard sekolah.

**Penyelidik (Peringkat cadangan Doktor Falsafah)**

Gunakan Prompt teroka topik untuk cari Research Gap sebenar, elak ulang dengan tesis dah ada di Dspace universiti besar di Malaysia.

**Pakar Penyelidikan Pasaran**

Gunakan teknik serupa untuk laporan industri, gunakan Prompt analisis data untuk terangkan nombor daripada Jathirdtan Perangkaan atau laporan kewangan syarikat.

## [Bandingan](thesis-writing-prompts.html#comparisahajan)

Kriteria| Tulis tradisional| Guna AI tak ada Prompt| Guna Prompt Pakar
---|---|---|---
Masa cari dokumen| 3-4 minggu| 3-4 hari (tapi mudah terpesahajang)| 5-7 hari (ada kawalan)
Sifat akademik| Tinggi (jika ada pengalaman)| Rendah (gaya harian)Tinggi (diedit pakar mendalam)
Risiko plagiar| Rendah| Tinggi (jika copy-paste)Rendah (AI cuma sahajakorang struktur)
Kesan ralat logic| Sukar sendiri sangat| SukawalSenang melalui Prompt kritikan

Prompt pakar memastikan gabung kelajuan AI dengan kualiti kaedah tradisional, teristimewa dalam tambahagian kawalan kualiti kandungan sebelum pembelaan.
