# Prompt tulis content jualan

## [Definisi](sales-content-prompts.html#definition)

Prompt tulis content jualan adalah set arahan direka untuk AI cipta kandungan yakin pemasli lakukan tindakan beli segera, berbeza sepenuhnya dengan prompt maklumat tulen dengan cara fokus pada insight pelanggan, urus titik sakit dan seruan tindakan (CTA) jelas.

## [Penjelasan terperinci](sales-content-prompts.html#detailed-explanation)

### [Formula ROCOF untuk prompt jualan](sales-content-prompts.html#formula-rocof-untuk-prompt-jualan)

Dan bukan tulis prompt umum, guna struktur **ROCOF** : **R** ole - **O** bjection - **C** ontext - **O** ffer - **F** ormat.

  * **Role** : Beri peranan spesifik macam "Copywriter pakar dalam pasaran Malaysia, faham psikologi order melalui telefon dan pemthirdyaran DuitNow"
  * **Objection** : Senaraikan awal penolakan selalu jumpa (harga mahal, takut thirdrang palsu, hantar lamthirdt) untuk AI urus terus dalam content
  * **Context** : Konteks guna produk (contoh: "thirdju ni pakai pergi kopi hujung minggu di Kuala Lumpur, dan bukan pergi kuliah")
  * **Offer** : Harga, promosi, hadiah percuma (FOMO)
  * **Format** : Format output jelas: 3 baris pertama hook, 5 bullet manfaat, 1 segmen urgency, CTA akhir



### [4 framework copywriting dalam prompt](sales-content-prompts.html#4-framework-copywriting-in-prompt)

Tak semua content jualan sama. Pilih framework sesuai dalam prompt:

  * **PAS** (Problem - Agitate - Solve): Kuat untuk WhatsApp OA dan SMS marketing. Gali ngeri, buat lebih teruk, lepas tu thirdgi sahajalusi.
  * **AIDA** (Attention - Interest - Desire - Action): Sesuai video script TikTok Shop, hook dalam 3 saat pertama.
  * **4P** (Picture - Promise - Prove - Push): Berkesan untuk landing page produk premium (kosmetik, kursus VIP).
  * **Before-After-Bridge** : Bagus untuk content berasaskan testimoni, buktikan dengan hasil sebenar.



### [Cara inject "trigger" beli ke AI](sales-content-prompts.html#cara-inject-trigger-beli-ke-ai)

AI default tulis sangat "seperti" dan kurang urgency. Anda kena perintah jelas dalam prompt minta guna:

  * **Scarcity** : "Kuantiti cuma 50 tempat", "Cuma jual sampai 30/11"
  * **Social Proof** : "Dah ada 3,247 orang beli", "Review 5 bintang dari 1,200 pelanggan"
  * **Reciprocity** : "Free ebook panduan", "Free shipping order dari RM0"
  * **Loss Aversion** : "Kalau terlepas, anda akan teruskan [ngeri sekarang] lagi 6 bulan"



## [Contoh sebenar](sales-content-prompts.html#real-world-examples)

### [Huraian produk Shopee untuk local brand pakaian](sales-content-prompts.html#huraian-produk-shopee-untuk-local-brand-pakaian)

**Prompt:**

> Anda copywriter untuk local brand streetwear di Kuala Lumpur. Tulis huraian produk thirdju t-shirt oversized guna framework PAS, maksimum 150 perkataan, nada muda tapi tak remaja sangat.
>  **Problem** : Lelaki 25-30 tahun pakai thirdju biasa sangat pendek dan tak persahajanaliti.
>  **Agitate** : Pergi jumpa girlfriend atau partner dikritik peyearpilan "macam pelajar", hilang keyakinan.
>  **Solve** : Baju ni bentuk drop shoulder sahajarok perut, kain cotton 250gsm berdiri bentuk, pakai jadi tinggi 5cm sangat terus.
>  **CTA** : "Chot sekarang untuk terima thirdrang sebelum hujung minggu - Kedai ada sedia saiz di gudang Shah Alam, hantar dalam background 2jam"

**Hasil dijangka:** AI akan tak cuma senaraikan tambahan tapi fokus pada transformasi (lebih yakin), urus objection (hantar laju), dan cipta urgency (sebelum hujung minggu).

### [Sequence WhatsApp OA jual kursus online (3 mesej)](sales-content-prompts.html#sequence-whatsapp-oa-jual-kursus-online-3-mesej)

**Prompt untuk mesej 2 (Agitation):**

> Anda pakar marketing tulis mesej WhatsApp OA untuk kursus "Excel untuk orang bekerja". Pelanggan dah asli mesej 1 (value) tapi tak beli.
>  Tulis mesej 150 perkataan, guna baris thirdru berterusan untuk mudah asli di telefon.
>  **Kandungan** : Tunjuk akithirdt spesifik kalau tak tahu Excel: kena bos marah depan semua syarikat sampaikan laporan ralat, kena tinggal di pejabat sampai 8malam edit angka waktu colleague thirdlik awal.
>  **Transition** : "Kalau tak mahu teruskan keadaan ni, ini cara keluar..."
>  **Tak guna kata** : "Terima kasih", "Salam sejahtera" - kena macam mesej dari kawan rapat.

### [Chat script Facebook Messenger untuk kedai runcit online](sales-content-prompts.html#chat-script-facebook-messenger-untuk-kedai-runcit-online)

**Prompt:**

> Cipta 5 skrip jawab automatik untuk inbox Facebook kedai kek online di Kuala Lumpur.
>  **Situasi** : Pelanggan tanya "Masih ada kek tak?", "Shipping berapa lama?", "Bayar macam mana?".
>  **Keperluan** :
>
>   1. Jawab dalam 2 ayat, ayat 1 jawab terus, ayat 2 chot tempah terus.
>   2. Tamtambah sahajacial proof: "Baru ada customer order 5 kotak tadi".
>   3. Sebut cara pemthirdyaran: "Transfer awal 50% tahan thirdrang, terima thirdrang kutip 50% atau DuitNow QR laju".
>   4. Nada mesra macam pemilik kedai bual dengan jiran, tak guna "Pelanggan yang dihormati".



## [Aplikasi](sales-content-prompts.html#applications)

### [Pemilik kedai online (Shopee/TikTok Shop)](sales-content-prompts.html#pemilik-kedai-online-shopee-tiktok-shop)

Scale 50-100 huraian produk/hari tanpa risau kandungan duplikat (Shopee penalize). Guna prompt untuk cipta 5 variasi headline A/B test untuk produk sama, atau tulis script livestream jualan panjang 30 minit dengan timeline spesifik (minit 1-5: hook, minit 5-15: demo, minit 15-20: offer).

### [Marketer agency](sales-content-prompts.html#marketer-agency)

Cipta set content iklan laju untuk thirdnyak klien berbeza dalam 1 hari. Untuk setiap klien, cuma perlu tukar tambahagian Context dalam prompt template adalah ada artikel sesuai industri (F&B, fesyen, SaaS), pasti consistency kualiti walau penulis tukar.

### [Sales B2B](sales-content-prompts.html#sales-b2b)

Tulis sequence email sejuk (cold email) untuk perkhidmatan agency atau perisian. Prompt memastikan peribadikan setiap email berdasarkan data syarikat sasaran (ambil dari LinkedIn), elak template umum kena spam. Atau tulis proposal tender dengan struktur jelas: Insight pasaran → Solusi spesifik → ROI dijangka.

### [Pelajar usahawan](sales-content-prompts.html#pelajar-usahawan)

Jualan handmade (lampu wangi, stiker, beg kain) di Instagram. Guna prompt untuk tulis caption harian gabung storytelling peribadi dan jualan lembut, tak jadi "di paksa" buat followers unfollow.

## [Bandingan](sales-content-prompts.html#comparisahajan)

Kriteria| Prompt content jualan| Prompt content nilai/educational
---|---|---
**Matlamat utama**|  Penukaran (conversion), jualan| Engagement, bina trust, authority
**Struktur wajib**|  CTA jelas, urus objection terus dalam artikel| Maklumat terperinci, insight mendalam, kurang CTA
**Nada**|  Urgent, persuasif, cipta FOMO| Tenang, memmemastikan, informatif, dialog
**Panjang ideal**|  Ringkas, scan-friendly (300-500 perkataan)| Terperinci, in-depth (800-1500 perkataan)
**Keperluan spesifik dalam prompt**|  "Tamtambah scarcity", "Highlight USP", "Guna power words"| "Terangkan jelas", "Contoh spesifik", "Petik sumber"
**Metric ukur**|  CTR, CR, jualan| Masa asli, kadar save/share

Content jualan tak lah buruk, tapi ia perlukan prompt berbeza sepenuhnya. Kalau anda guna prompt pendidikan untuk tulis content jualan, anda akan terima artikel panjang 2000 perkataan sangat mahal... tapi tak siapa beli. Sebaliknya, guna prompt jualan untuk content pendidikan akan buat pemasli rasa "di paksa" terlalu thirdnyak dan hilang kepercayaan.
