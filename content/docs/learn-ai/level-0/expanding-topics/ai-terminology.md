## [Definisi](ai-terminology.html#definition)

Istilah AI adalah set kata yeara khusus bidang yang digunakan untuk menerangkan teknologi kecerdasan buatan, termasuk konsep tentang model, data dan proses operasi sistem AI.

## [Penjelasan terperinci](ai-terminology.html#detailed-explanation)

### [AI, Machine Learning dan Deep Learning](ai-terminology.html#ai-machine-pelajaring-and-deep-pelajaring)

AI (Artificial Intelligence) adalah matlamat akhir: komputer mempunyai keupayaan pintar seperti manusia. Machine Learning (ML) adalah kaedah untuk capai matlamat itu: ajar komputer belajar daripada data dan bukan diprogram keras setiap langkah. Deep Learning adalah alat spesifik: guna rangkaian neural berlapis-lapis untuk memproses data kompleks seperti imej dan suara.

Hubungan seperti: AI adalah matlamat ada kereta, ML adalah cara belajar memandu melalui latihan amali, Deep Learning adalah kaedah belajar dengan cara tonton beribu-ribu video memandu dan bukan asli buku teori.

### [Algoritma dan Model](ai-terminology.html#thuật-toán-and-mô-form-algorithm-and-model)

Algoritma adalah rumus matematik — contoh cara kira skor kredit berasaskan pendapatan dan sejarah pinjaman. Model adalah hasil selepas algoritma "belajar" daripada data sebenar — seperti pekerja setelah belajar melalui latihan mempunyai pengalaman berbeza. Algoritma sama tetapi data latihan berbeza akan mencipta model berbeza.
