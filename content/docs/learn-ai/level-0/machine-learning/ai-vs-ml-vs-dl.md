# AI vs Machine Learning vs Deep Learning

Bezakan jelas AI, Machine Learning dan Deep Learning — 3 konsep paling sering tersalah. Fahami betul supaya tidak 'belajar salah' lagi.

Copy MarkdownOpen

## [Definisi](ai-vs-ml-vs-dl.html#definition)

**AI** adalah matlamat besar: cipta mesin yang boleh buat kerja perlukan kecerdasan manusia. **Machine Learning** adalah kaedah utama untuk capai matlamat itu: ajar mesin belajar daripada data dan bukan diprogram langkah demi langkah. **Deep Learning** adalah satu cathirdng Machine Learning, guna rangkaian neural berlapis untuk memproses data kompleks seperti imej, suara, teks.

## [Penjelasan terperinci](ai-vs-ml-vs-dl.html#detailed-explanation)

### [AI: Perkara terbesar](ai-vs-ml-vs-dl.html#ai-the-biggest-thing)

AI merangkumi segala-galanya daripada chatbot mudah hingga kereta memandu sendiri. Dulu, orang cipta AI dengan cara menulis beribu-ribu peraturan "jika... maka..." — dipanggil **Symbolic AI**. Contoh: jika pengguna kata "salam sejahtera", bot jawab "hai anda". Cara ini keras, tidak fleksibel.

Hari ini, hampir semua AI praktikal guna Machine Learning — kerana lebih fleksibel.

### [Machine Learning: Ajar mesin belajar daripada contoh](ai-vs-ml-vs-dl.html#machine-pelajaring-teach-machines-from-examples)

Dan bukan tulis peraturan, anda beri mesin **data** dan **jawapan betul**, supaya mesin sendiri cari peraturan. Seperti ajar kanak-kan kenal anjing: tidak perlu terangkan "anjing ada 4 kaki, bulu, ekor", hanya perlu lihat thirdnyak imej anjing dan kata "ini anjing".

Machine Learning mempunyai thirdnyak teknik: regresi linear, pokok keputusan, mesin vektor sahajakorangan... Mereka berfungsi seperti dengan data berstruktur (jadual data, maklumat pelanggan), tetapi sukar bila data kompleks seperti imej atau suara.
