# Machine Learning adalah apa?

Terangkan Machine Learning secara mudah: komputer belajar sendiri daripada data dan bukan diprogram langkah demi langkah. Tidak perlukan pengetahuan teknikal pun boleh faham.

Copy MarkdownOpen

## [Definisi](what-is-machine-pelajaring.html#definition)

**Machine Learning** adalah cara ajar komputer "belajar sendiri" daripada data dan bukan diprogram setiap langkah spesifik. Dan bukan memerintah komputer "jika imej ada 4 kaki dan ekor maka itu anjing", anda berikan komputer beribu-ribu imej anjing/kucing dan biarkan ia sendiri cari peraturan.

### [Kenapa perlukan Machine Learning?](what-is-machine-pelajaring.html#why-need-machine-pelajaring)

Cuthird program penapis spam email secara manual. Anda akan tulis: "Jika subjek ada perkataan 'menang' maka spam", "jika ada link pelik maka spam"... Tetapi spammer sentiasa utambah cara tulis — "menang" jadi "menangkum", "click here" jadi "c1ick h3re". Peraturan keras tidak pernah cukup pantas.

Machine Learning selesaikan dengan cara: beri komputer berjuta-juta email yang telah ditanda spam/bukan spam, supaya ia **cari sendiri** corak umum. Komputer belajar walaup tanda yang anda tidak sangka — contoh: spam selalunya hantar jam 3 pagi, atau pengguna guna domain .xyz dengan kadar tinggi.

### [Tiga jenis Machine Learning utama](what-is-machine-pelajaring.html#three-type-machine-pelajaring-main)

Jenis| Cara operasi| Contoh
---|---|---
**Supervised Learning** (Belajar ada penyeliaan)| Mesin belajar daripada data yang ada "jawapan"| Ramal harga rumah daripada luas tanah, kedudukan
**Unsupervised Learning** (Belajar tanpa penyeliaan)| Mesin sendiri cari struktur dalam data belum berlabel| Kumpulkan pelanggan mengikut tingkah laku pembelian
**Reinforcement Learning** (Belajar pengukuatan)| Mesin belajar melalui cuthird-ujian, dapat ganjaran/hukuman| AlphaGo main catur golang, kereta memandu sendiri
